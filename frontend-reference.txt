Frontend Structure Reference (REFACTORED)
======================================

Root Files:
-----------
.env                                  - Environment variables
.gitignore                             - Git ignoring rules
components.json                       - UI component configuration
eslint.config.mjs                     - ESLint configuration
google-indexing-credentials.json      - Google Indexing API credentials
index-images.sh                       - Script for indexing images
indexnow-payload.json                 - IndexNow API payload
next.config.ts                        - Next.js configuration
next-env.d.ts                         - Next.js TypeScript environment
next-sitemap.config.js                - Sitemap generation configuration
package.json                          - Project dependencies and scripts
package-lock.json                     - Locked dependencies
postcss.config.mjs                    - PostCSS configuration
README.md                             - Project documentation
tailwind.config.ts                    - Centralized theme configuration (Tokens: shadow, blur, radius, glass)
tsconfig.json                         - TypeScript configuration
verify-indexing-setup.sh              - Script to verify indexing setup

Folders:
--------
public/                               - Static assets (images, icons, etc.)
src/                                  - Source code

src/ Structure:
---------------
actions/                              - Server actions
app/                                  - Next.js App Router pages and layouts
components/                           - Reusable components
    Provider/                         - Context providers
        ThemeProvider.tsx             - Type-safe dark/light mode management
    atomic/                           - Atomic design components
        atoms/                        
            Icon.tsx                  - Unified Lucide-React wrapper atom
            ChatAvatar.tsx, etc.
        molecules/
    features/                         - Feature-specific components
    layout/                           - Navbar, Footer
    ui/                               - Base UI components (Refactored for A11Y & Tokens)
        Button.tsx                    - Type-safe variant-based button
        Modal.tsx                     - Headless-ready accessible modal
        Card.tsx, etc.
config/                               - Configuration files
data/                                 - Static data
hooks/                                - Custom React hooks
lib/                                  - Library code and utilities
    utils.ts                          - Tailwind-merge and general utils
mocks/                                - Mock data
services/                             - Data fetching and API services
store/                                - Redux state management
types/                                - TypeScript type definitions
proxy.ts                              - Proxy configuration
